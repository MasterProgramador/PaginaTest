Mi primera pagina en Visual Studio Code

Estoy usando VS (Visual Studio).

Ventajas de VS a comparacion de Eclipse:

  • Asistente IA
  • Autocompletado
  • Compatible con muchos lenguajes
  • Codigo abierto y de Windows
  • Mejor marcado de errores
  • Una lista mas extensa de extensiones
  • Interfaz intuitiva

Hasta ahora, me parece que es mucho mas util VS a Eclipse.
Ademas de que me es util para mi curso de Desarrollo Web.

Aun me falta bastante por aprender de HTML, CSS y JavaScript.
Pero hasta ahora voy bien encaminado.
